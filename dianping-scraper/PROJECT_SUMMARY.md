## 使用方式

### 方式1: 基本使用

```python
from scraper import DianpingScraper

scraper = DianpingScraper(city="北京", keyword="火锅")
merchants = scraper.search_merchants("北京", "火锅", page=1)
scraper.export_to_csv('beijing_hotpot.csv')
```

## 配置说明

所有配置都集中在 `config.py` 文件中，包括：

- **REQUEST_CONFIG**: 请求超时、重试次数等
- **DELAY_CONFIG**: 延时时间范围
- **USER_AGENTS**: User-Agent 列表
- **PROXY_CONFIG**: 代理设置
- **EXPORT_CONFIG**: 导出格式和目录
- **PLOT_CONFIG**: 图表样式配置
- **LOG_CONFIG**: 日志设置

## 注意事项

1. **遵守网站规则**: 使用时请遵守大众点评的 robots.txt 和服务条款
2. **请求频率**: 不要过度频繁请求，使用内置延时功能
3. **数据用途**: 数据仅用于学习和研究，不得用于商业用途
4. **网站结构**: 网站结构可能变化，需要及时更新解析逻辑
5. **反爬机制**: 可能遇到反爬，需要调整延时或使用代理
