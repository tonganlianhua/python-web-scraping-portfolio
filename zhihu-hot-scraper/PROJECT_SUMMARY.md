# 知乎热榜爬虫 - 项目总结

## 项目概述

这是一个功能完整的知乎热榜爬虫项目，实现了从获取热榜、数据分析到可视化的完整工作流。

## 已完成功能

### ✅ 核心功能

1. **实时获取知乎热榜**
   - 支持获取当前排名前50的热门话题
   - 包含排名、标题、热度值、链接、摘要等信息
   - 自动记录采集时间

2. **关键词过滤**
   - 支持按关键词筛选话题
   - 不区分大小写匹配

3. **话题详情获取**
   - 获取回答数、关注数、浏览量
   - 获取话题标签
   - 自动处理解析错误

4. **多次采集与热度对比**
   - 支持多次采集热榜
   - 自动计算热度变化
   - 识别新上榜和已下榜话题
   - 计算变化百分比

5. **Excel 导出**
   - 导出热榜数据到格式化的 Excel
   - 导出热度对比结果
   - 导出话题详情
   - 自动设置列宽和样式

6. **趋势分析**
   - 生成热度趋势折线图
   - 生成热度变化对比条形图
   - 支持中文显示

### ✅ 反爬策略

1. **User-Agent 轮换** - 内置7个常见 User-Agent
2. **随机延时** - 每次请求前随机延时 0.5-2 秒
3. **失败重试** - 最多重试 3 次
4. **限流处理** - 检测 429 状态码并自动调整
5. **Session 复用** - 复用连接提高效率

## 使用方式

### 方式1: 快速开始
```bash
python quick_start.py
```

### 方式2: 交互式演示
```bash
python demo.py
```

### 方式3: 使用示例集
```bash
python example_usage.py
```

### 方式4: 定时监控
```bash
python monitor.py once  # 单次采集
python monitor.py monitor --interval 10  # 循环监控
python monitor.py batch --times 5  # 批量采集
```

### 方式5: 运行测试
```bash
python test_scraper.py
```

### 方式6: 作为模块使用
```python
from zhihu_scraper import ZhihuHotScraper
from data_exporter import DataExporter

scraper = ZhihuHotScraper()
hot_list = scraper.get_hot_list()
exporter = DataExporter()
exporter.export_to_excel(hot_list)
```
